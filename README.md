# gameconsign_interview
This is repo for interview with gameconsign

## STEP TO USE

1. install npm with `npm install`
2. if want to run with change run code with `npm run dev` if deploy for production just use `npm run start`
3. this application default running on http://localhost:3000

## POSTMAN LINKS
for POSTMAN collection can access [here](https://www.getpostman.com/collections/aa65a611484ee857580b) 

## DETAILS PAGE
1. Page anagram use post method and data sample can check on POSTMAN collection  http://localhost:3000/anagram
2. Page plus will use like this one http://localhost/plus/3

4. For API section you can check sample on collection POSTMAN , we will use link http://localhost:3000/api
